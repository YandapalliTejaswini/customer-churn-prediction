# Customer Churn Prediction

## Problem
Predict customers likely to leave telecom service.

## Tech Stack
Python, Pandas, Scikit-learn, Streamlit

## Steps
- Data Cleaning
- EDA
- Random Forest Model
- Feature Importance
- Dashboard Deployment

## Model Accuracy
~85%

## Business Impact
Helps company reduce customer loss.

## Run Project
python train.py
streamlit run app.py
